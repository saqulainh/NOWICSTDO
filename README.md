# NowicStdo

Premium, modern, fully responsive software agency website for **Nowic Studio**.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide Icons
- React Router

## Pages

- Home
- Services
- Portfolio
- About
- Contact

## Run Frontend

1. Go to frontend folder:
	- `cd frontend`
2. Install dependencies:
	- `npm install`
3. Start dev server:
	- `npm run dev`

## Build

- `npm run build`
